# Final Project - Call Center KPI Dashboard Analysis

## Project Overview

### Dataset Description

The Call Center details dataset contains information on call ID, agent, date, time, topic, calls answered, calls resolved, speed of answer in seconds, avg talk duration and satisfaction rating. The dataset is focused on customer service activities.

### Objectives (Business problem):
* Analyze how many calls were answer during the week.
* Idetify the satisfaction ratings.
* Identify speed of answers made by different agents.
* Analyze the overall resolution rate as well as the one done individually by each agents.
* Determine the amount of inbound calls each day as well as the duration of the call.
* Identify the number of resolved calls by each individual agent.

### Data Preparation (data cleaning, structure, and model):

* Load your data from from different sources and establishes the foundation for your data model.
* Combine Data (Joins, Relationships, Blends)
* Clean & Transform Data; rename columns change data types, fix incorrect values or nulls, split or merge columns.
* Create Calculated Fields
* Apply Aggregations & Hierarchies

### Data View 
* Displayed the call volume during the week and their respective count using vertical bar chart.
* Displayed the satisfaction rating and its corresponding counts using a vertical bar chart.
* Showed the speed of answers done by each agents in seconds using a highlight table.
* Displayed the resolution rates done by each agent using a highlight table.
* Displayed the resolved calls done by each agent and their respective counts using a highlight table.
* Showed the overall resolution rate using a card.
* Showed the inbound calls of the day and its duration in seconds using a card.
